## 一、下载与安装

### Windows

1. 下载 `Agentrix-Desktop_0.1.0_x64-setup.exe` 安装包
2. 双击运行安装程序
3. 如果看到 Windows SmartScreen 提示，点击 **"更多信息"** → **"仍要运行"**
4. 按照安装向导完成安装
5. 安装完成后，Agentrix 会自动启动

> **提示**: 安装包仅约 4 MB，安装速度非常快。

### macOS

1. 下载 `Agentrix-Desktop_0.1.0_aarch64.dmg` (Apple Silicon) 或 `_x64.dmg` (Intel)
2. 双击打开 DMG 文件
3. 将 Agentrix 拖到 Applications 文件夹
4. 首次打开时，如果提示"不受信任的开发者"：
   - 打开 **系统设置** → **隐私与安全** → 点击 **"仍要打开"**
5. 在 Applications 中双击启动

### Linux

1. 下载 `Agentrix-Desktop_0.1.0_amd64.AppImage`
2. 添加执行权限:
   ```bash
   chmod +x Agentrix-Desktop_0.1.0_amd64.AppImage
   ```
3. 双击运行，或在终端执行:
   ```bash
   ./Agentrix-Desktop_0.1.0_amd64.AppImage
   ```

## 二、首次启动与登录

### 启动后界面

首次启动后，你会看到登录界面。Agentrix 提供 4 种方式进入：

### 方式一：📱 扫码登录（推荐）

如果你已经在手机上使用 Agentrix (ClawLink)：

1. 在登录页切换到 **📱 扫码** 标签页
2. 打开手机 App → 扫描屏幕上的二维码
3. 在手机上确认授权
4. 桌面端自动登录成功

### 方式二：📧 邮箱登录

1. 切换到 **📧 邮箱** 标签页
2. 输入你的邮箱地址
3. 点击 **Send Verification Code**
4. 在邮箱中找到 6 位验证码，输入并确认
5. 登录成功

### 方式三：🔗 第三方登录

1. 切换到 **🔗 第三方** 标签页
2. 点击 **Continue with Google** 或 **Continue with Discord**
3. 系统浏览器会自动打开对应平台的授权页面
4. 在浏览器中完成授权
5. 回到桌面端，自动获取登录状态

### 方式四：访客体验

不想注册？点击底部的 **Skip as Guest →** 即可以访客身份体验基础功能。

> **注意**: 访客模式下部分功能受限（如历史记录同步、跨设备协同）。

### 首次引导

登录后会进入简短的引导流程，介绍核心功能：
1. **欢迎页** — 了解 Agentrix 的核心能力
2. **连接设置** — 选择 AI 实例（可跳过）
3. **快捷键** — 了解全局热键

完成引导后，你就可以开始使用了！

## 三、悬浮球操作

登录完成后，屏幕上会出现一个紫色的悬浮球 — 这是你与 AI 交互的入口。

### 基本操作

| 操作 | 效果 |
|------|------|
| **单击** | 打开/收起对话面板 |
| **长按** | 进入语音模式（按住说话，松开发送） |
| **右键** | 弹出快捷菜单 |
| **拖拽** | 移动悬浮球位置 |

### 拖拽与吸附

- 按住悬浮球不放，可以拖到屏幕任意位置
- 松手后，悬浮球会自动吸附到最近的屏幕边缘
- 位置会被记住，下次启动时恢复

### 透明度变化

- 鼠标悬停时：100% 不透明
- 正常状态：80% 不透明
- 空闲 30 秒后：自动降低透明度（不遮挡工作）
- 点击或悬停立即恢复

### 右键菜单

右键点击悬浮球，弹出快捷菜单：

| 选项 | 功能 |
|------|------|
| 🔄 新对话 | 开始一个全新的对话 |
| ⚙️ 设置 | 打开设置面板 |
| 🤖 切换模型 | 切换 AI 模型 |
| ❌ 退出 | 完全退出应用 |

### 首次引导气泡

如果你是第一次使用，悬浮球上方会显示一个提示气泡：**"点击我开始对话 👋"**。点击悬浮球即可消除提示，开始正常使用。

## 四、对话聊天

### 开始对话

1. 单击悬浮球，展开对话面板
2. 在底部输入框中输入你的问题
3. 按 **Enter** 发送
4. AI 会以流式方式逐字回复

### 消息格式

AI 的回复支持丰富的格式：

- **Markdown 文本** — 标题、加粗、斜体、列表
- **代码块** — 自动语法高亮（支持 Python、JavaScript、Rust 等 20+ 语言），右上角有 📋 **复制** 按钮
- **思维链** — AI 的思考过程，默认折叠，点击可展开查看

### 发送文件

- 点击输入框旁的 📎 **附件按钮**
- 或直接将文件 **拖拽** 到对话面板
- 支持的文件类型：文本、代码、图片、PDF

### 多行输入

- **Enter** — 发送消息
- **Shift + Enter** — 换行（不发送）

### 会话管理

- **新建会话**: 点击面板顶部的 ➕ 按钮，或按 `Ctrl+N`
- **历史记录**: 面板左侧有历史会话列表，点击可切换
- **清除当前会话**: 输入 `/clear` 命令

### 后台通知

当你切到其他窗口工作时，如果 AI 完成了回复，会收到桌面系统通知提醒你查看。

## 五、语音交互

### 使用方式

| 方式 | 操作 |
|------|------|
| **长按悬浮球** | 按住开始录音，松开发送 |
| **全局热键** | 按住 `Ctrl+Shift+A` 录音，松开发送 |
| **面板内** | 点击输入框旁的 🎤 按钮 |

### 语音对话流程

1. 按住录音 → 悬浮球显示录音动画
2. 松开 → 语音自动发送到服务器转文字
3. AI 处理你的问题 → 文字回复出现在面板中
4. 如果开启了 TTS，AI 回复会自动朗读

### 语言支持

- **语音识别**: 自动检测中文/英文
- **语音合成**: 中文使用智雨 (Zhiyu)、英文使用 Matthew

### TTS 语音播放

- 默认开启，可在 **设置** 中关闭
- AI 回复会自动分段朗读
- 播放过程中可以继续发送新消息

## 六、斜杠命令

在对话输入框中输入 `/` 开头的命令，可以快速执行特定操作：

| 命令 | 功能 | 示例 |
|------|------|------|
| `/help` | 显示所有可用命令 | `/help` |
| `/search <关键词>` | AI 联网搜索 | `/search 最新AI新闻` |
| `/skill` | 查看可用技能列表 | `/skill` |
| `/skill <名称>` | 激活指定技能 | `/skill translator` |
| `/model <名称>` | 切换 AI 模型 | `/model claude` |
| `/new` | 新建对话 | `/new` |
| `/clear` | 清空当前对话 | `/clear` |
| `/ls [路径]` | 列出工作区目录 | `/ls src/` |

## 七、文件与工作区

Agentrix 可以读写你电脑上的文件，帮你分析代码、修改文档。

### 设置工作区

1. 打开 **设置** (右键悬浮球 → 设置，或按 `Ctrl+Shift+S` 后点击设置图标)
2. 点击 **选择工作区** 按钮
3. 选择你的项目文件夹
4. 选定后，AI 就可以读取和修改该文件夹中的文件了

### 在对话中使用

设置工作区后，你可以在对话中这样说:

- "帮我看一下 src/main.ts 这个文件"
- "把 README.md 的标题改成 Agentrix"
- "分析整个 src 目录的代码结构"

AI 会自动调用文件读写能力来完成你的请求。

### 快速浏览

输入 `/ls` 可以列出工作区的文件:
```
/ls            ← 列出根目录
/ls src/       ← 列出 src 子目录
```

> **安全**: 文件操作仅限于你选定的工作区文件夹内，AI 无法访问电脑上的其他位置。

## 八、设置与偏好

### 进入设置

- 右键悬浮球 → **⚙️ 设置**
- 或在面板中点击设置图标

### 可用选项

| 设置项 | 说明 |
|--------|------|
| **检查更新** | 手动检查是否有新版本 |
| **工作区路径** | 选择或更换项目文件夹 |
| **TTS 语音** | 开启/关闭 AI 回复朗读 |
| **开机自启** | 是否开机时自动启动 Agentrix |
| **退出应用** | 完全退出 Agentrix |

### 系统托盘

最小化后，Agentrix 会驻留在系统托盘 (Windows 通知区域 / macOS 菜单栏)。右键托盘图标可以:

- 显示/隐藏悬浮球
- 打开设置
- 退出应用

## 十、常见问题

### Q: 安装后 Windows 提示"不受信任"怎么办？

这是因为应用尚未购买 Windows 代码签名证书。点击 **"更多信息"** → **"仍要运行"** 即可正常安装。应用本身是安全的。

### Q: macOS 无法打开怎么办？

首次打开可能被 Gatekeeper 阻止：
1. 打开 **系统设置** → **隐私与安全性**
2. 找到 "已阻止 Agentrix Desktop" 的提示
3. 点击 **"仍要打开"**

### Q: 全局热键没有反应？

- 确认 Agentrix 正在运行 (检查系统托盘)
- 某些应用可能占用了相同的快捷键组合
- 可在设置中查看当前快捷键配置

### Q: 语音录制没有声音？

1. 检查麦克风是否正确连接
2. 确认浏览器/系统已授予 Agentrix 麦克风权限
3. Windows: **设置** → **隐私** → **麦克风** → 确认已开启

### Q: 如何更新到最新版本？

- 应用会自动检查更新
- 手动检查: 右键悬浮球 → **设置** → **检查更新**
- 有新版本时会弹窗提示，确认后自动下载安装

### Q: 访客模式和登录有什么区别？

| 功能 | 访客 | 登录用户 |
|------|------|---------|
| 基础对话 | ✅ | ✅ |
| 语音交互 | ✅ | ✅ |
| 历史同步 | ❌ | ✅ |
| 跨设备协同 | ❌ | ✅ |
| 技能市场 | ❌ | ✅ |
| OpenClaw 实例 | ❌ | ✅ |

### Q: 数据存储在哪里？

- **认证 Token**: OS 密钥链加密存储 (Windows Credential Manager / macOS Keychain)
- **对话历史**: 浏览器 localStorage (本地)
- **设置偏好**: 本地存储
- **文件操作**: 仅在你选定的工作区文件夹内

### Q: 如何完全退出？

- 右键悬浮球 → **❌ 退出**
- 或右键系统托盘图标 → **退出**
- 关闭面板只是收起，不会退出应用

### Q: 支持哪些 AI 模型？

默认使用 Claude (Anthropic)。登录后可在对话中输入 `/model` 切换，支持的模型取决于你绑定的 OpenClaw 实例配置。常见选项：

- Claude 3.5 Sonnet / Opus
- GPT-4o
- Gemini Pro
- DeepSeek
- Groq (Llama)

### Q: 如何反馈 Bug 或建议？

- 在对话中告诉 AI: "我要反馈一个 bug"
- 或通过 Agentrix 社区渠道提交
