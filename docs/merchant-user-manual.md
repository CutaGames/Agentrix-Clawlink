# 商户端用户手册（简明）

面向商户用户的简明操作手册，列出当前可用的主要功能、页面路径与常用操作步骤。

**导航与入口**
- 登录后进入商户后台：`/app/merchant`（仪表盘）。
- 常见菜单项：`数据概览`、`商品管理`、`订单管理`、`提现管理`、`商户信息`、`KYC`、`自动化`。

**功能一览（面向商户）**
- **数据概览** (`/app/merchant`)：查看总销售额、AI 渠道销售、订单数量、AI Agent 数量与最近订单。适合日常监控与趋势查看。
- **商品管理** (`/app/merchant/products`)：
  - 查看已上架/下架商品列表。
  - 支持搜索、筛选与跳转到商品详情/编辑。
  - 注意：外部电商平台数据（如图片、SKU、原始价格）保存在商品的 `metadata`，在编辑/展示时会从 `metadata` 读取。
- **批量导入** (`/app/merchant/batch-import`)：上传 CSV/Excel 文件批量创建/更新商品。上传后系统会处理并返回导入结果与错误行反馈。
- **电商平台同步** (`/app/merchant/ecommerce-sync`)：
  - 管理与外部平台的同步映射（决定外部字段如何映射到本系统）。
  - 手动触发导入/同步操作。对于大规模同步，导入可能在后台完成，请留意同步状态或导入结果通知。
- **订单管理** (`/app/merchant/orders`)：查看商户所属订单、订单详情、交易状态与退款/售后记录。
- **商户信息 / 支付设置** (`/app/merchant/profile`)：配置商户名称、结算账户、提现方式与 API 密钥（如果已开放）。
- **自动化（自动接单 / AI 客服 / 自动营销）** (`/app/merchant/automation` 或在仪表盘侧边项)：
  - **自动接单**：设置自动接单策略（例如风控阈值、白名单、自动通过规则）。保存后系统会在符合规则的订单上自动处理接单逻辑。
  - **AI 客服**：与客户对话、查看历史会话、使用 AI 建议的操作（如推荐商品、发起退款、转人工）。当前为规则/Mock 支持，未来可接入真实 LLM。
  - **自动营销**：创建基于事件的营销规则（例如订单完成后推送优惠券）。
- **对账与结算** (`/app/merchant/reconciliation` 或在后台功能里)：定期查看对账记录、差异与未匹配交易，导出明细以便财务核对。

**常见工作流示例**
- 批量上新商品：进入 `批量导入` 页面 -> 下载模板 -> 填写 SKU/价格/库存 -> 上传 -> 等待导入结果 -> 在 `商品管理` 验证商品展示。
- 触发电商平台同步：进入 `电商平台同步` -> 配置字段映射 -> `开始同步` -> 检查同步日志或导入结果 -> 验证 `商品管理` 页面中 `metadata` 字段展示的图片与 SKU。
- 配置自动接单：进入 `自动化` -> 自动接单 -> 配置规则并启用 -> 观察后续符合条件订单是否自动被系统接单。

**注意事项与建议**
- 导入/同步为异步或耗时任务时，请在导入完成后核对结果；若发现字段缺失（如图片、SKU），请检查映射规则或 `metadata` 内容。
- 涉及财务/结算的操作（提现、结算规则）请与财务团队确认配置正确性，避免因配置失误导致资金异常。
- 当前 AI 功能可能使用内置规则或 Mock 实现；在使用 AI 自动处理关键业务（如退款或退单）前，建议先在小范围内观测效果并启用人工复核。

# Agentrix 商户使用手册（简明）

版本：2025-12-11

目标读者：平台商户（店主 / 运营）

快速起步
- 登录商户后台（示例路径）：`/app/merchant`。
- 常见页面：
  - 商品管理：`/app/merchant/products`
  - 批量导入：`/app/merchant/batch-import`
  - 电商平台同步：`/app/merchant/ecommerce-sync`

1) 商户界面总览
- 商品管理页面：查看、编辑、上架/下架商品，进入商品详情以编辑描述、价格、库存、图片和分润设置。
- 批量导入：用于一次性导入大量商品（CSV / JSON）。
- 电商同步：连接外部商店（Shopify / WooCommerce 等），自动或手动将外部商品同步到 Agentrix。

2) 添加单个商品（快速步骤）
1. 进入 `商品管理` → 点击 `添加商品`。  
2. 填写商品名称、分类、价格、库存、描述。  
3. 上传/粘贴图片 URL 到图片字段（建议至少一张主图）。  
4. 设置商品类型（实物 / 服务 / NFT / FT / 游戏资产 / RWA 等）与分润比例。  
5. 保存后如需上架，选择 `提交审核`（若平台设置审核流程）或直接上架（取决于权限）。

3) 批量导入（CSV / JSON）——详细操作指南
- 入口：`商品管理` → `批量导入`。
- 步骤：
  1. 点击 `下载模板`，下载 CSV 示例。模板包含示例列：`name, description, price, stock, category, productType, images (逗号分隔), tags`。
  2. 使用 Excel 或文本编辑器填写每行商品。图片列可以填图片 URL，多张用 `;` 或 `,` 分隔（以模板说明为准）。
  3. 回到导入页面，拖拽或选择 CSV 文件上传；页面会解析并展示预览表格，指出错误行与列。
  4. 检查预览：若有错误（如价格不是数字、必填缺失、图片 URL 格式不对），修正 CSV 后重新上传。
  5. 选择导入模式：
     - 创建（create）：仅新增新商品
     - 覆盖（overwrite）：根据外部 ID 或指定唯一键更新已有商品
     - 合并（merge/upsert）：存在则更新，不存在则创建
  6. 点击 `开始导入`，等待导入结果。导入结束后页面显示：导入数 / 更新数 / 失败数，并给出一小部分失败示例与原因。可下载失败记录以便修正后重试。

- 导入注意事项：
  - `price` 必须为数字，货币按平台配置（默认 CNY）。
  - `productType` 需在支持范围内：`physical, service, nft, ft, game_asset, rwa`。
  - 建议至少上传 1 张主图，图片需为可访问的 URL。

4) 电商平台同步（对接外部商店）
- 入口：`商品管理` → `电商平台同步`。
- 支持平台：Shopify、WooCommerce、Magento、BigCommerce、以及自定义 API。
- 对接步骤（以 Shopify 为例）：
  1. 点击 `添加连接` → 选择 `Shopify`。  
  2. 按需填写 `apiKey`, `apiSecret`, `storeDomain`（后台会给出具体凭证项说明）。  
  3. 创建连接：系统会保存连接并显示在连接列表。
  4. 点击 `同步`：触发一次手动同步；同步结果页面会显示“导入 / 更新 / 失败”统计与部分错误示例。
  5. 可启用自动同步（如果商户账户支持），系统会按设定间隔自动拉取并同步。

- 同步后数据规则：
  - 外部商品会在 Agentrix 创建为内部 `Product`，外部关键字段（images、sku、原价等）存放在 `metadata` 中。
  - 同步会创建映射记录（`ProductSyncMapping`），用于后续增量更新与冲突解决。

- 常见失败原因与排查：
  - 凭证错误或权限不足（返回 401/403）：请确认平台 API 权限、域名与凭证。  
  - 图片无法访问：检查图片 URL 是否可公网访问。  
  - 分类 mapping 未配置：如外部分类需映射，请联系平台管理员配置 `categoryMapping`。

5) 商品审核（如果平台启用）
- 当你提交商品审核：后台会自动运行一套基础自动审核规则（检查名称长度、描述、价格、图片等），给出评分与问题列表。  
- 状态说明（常见）：
  - `pending_review` / 待审核：已提交，需要管理员人工审核。  
  - `in_review` / 审核中：管理员正在处理。  
  - `approved` / 已通过：可上架并展示给用户（视平台策略）。  
  - `rejected` / 已拒绝：管理员拒绝，通常会附带拒绝原因。  
  - `revision_requested` / 请求修改：管理员要求修改特定字段，修改后需重新提交。

- 商户操作建议：
  - 遵循自动审核建议（如补充描述、增加图片、修正价格），这样通过率更高。
  - 若被拒绝，请根据拒绝原因修改并重新提交审核。

6) 管理与监控
- 查看导入与同步结果：在 `批量导入` 页面与 `电商平台同步` 页面查看统计；对失败的记录下载并修正后重试。
- 日志与支持：如遇持续性错误，请联系平台支持并提供失败样本与请求时间。

7) 常见问题（FAQ）
Q: 同步后为何商品图片未显示？  
A: 请检查外部图片 URL 是否可公网访问，或在导入时确保图片 URL 有效；也可以把图片先上传到你的图床或 Agentrix 图片托管。  

Q: 批量导入报价格错误怎么办？  
A: 确认价格列为纯数字（无货币符号），小数点使用 `.`。  

Q: 我如何撤销一次导入？  
A: 目前不支持全自动回滚，请联系管理员提供导入的批次信息，由管理员执行回滚或人工删除。建议先在测试环境验证 CSV。

8) 支持与联系方式
- 首选：平台后台工单或 `support` 页面提交工单（请附：商户ID、时间、失败示例文件/截图）。
- 紧急：联系运营/产品负责人（内部联系方式）
