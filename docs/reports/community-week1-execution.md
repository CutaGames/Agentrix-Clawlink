## 3. GitHub Community

### CONTRIBUTING.md

```markdown
# Contributing to Agentrix

Thank you for your interest in contributing! We welcome bug reports, feature requests, documentation, and code.

## How to Contribute

1. Fork the repository and create your branch from `main`.
2. Describe your changes clearly in your pull request.
3. For bug reports, include steps to reproduce and screenshots.
4. For feature requests, explain the use case and expected behavior.
5. Check for existing issues before opening a new one.
6. Follow our code style and add tests if applicable.

## Getting Help

- Join our [Discord](https://discord.gg/agentrix) or [Telegram](https://t.me/agentrix)
- For urgent issues, tag a maintainer in your issue or PR
```

### Good First Issues

1. **Improve onboarding documentation** — Update README with clearer setup steps
2. **Add dark mode toggle to web app** — Implement dark mode toggle button
3. **Fix typo in user profile page** — "Nickame" → "Nickname"
4. **Add unit tests for login component** — Improve test coverage
5. **Create GitHub Actions workflow for linting** — Auto-lint on PRs
