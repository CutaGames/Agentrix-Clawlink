### 2. 审批与 Agent 管理入口优化

**现状分析** (移动端已有能力):

| 功能 | 当前位置 | 问题 |
|------|---------|------|
| 审批事项 | Agent 控制台 → 桌面控制 (DesktopControlScreen) | 入口太深，仅支持桌面命令审批 |
| Agent 活动日志 | Agent 控制台 → 日志 (AgentLogsScreen) | 有通知 Feed（任务/错误/收益等），30s 刷新 |
| 团队空间 | Agent 控制台 → 团队空间 (TeamSpaceScreen) | 支持创建工作区、成员管理、角色分配 |
| Agent 协作 | Agent 控制台 → 空间 (AgentSpaceScreen) | 任务房间、@Agent 提及 |

**创始人需求**: 在移动端快速查看所有 Agent 的进展、审批待办、给 Agent 提意见。

**建议优化方向** (作为 P1 研发任务):
- 主 Tab 增加 "团队" 入口（或在 "Me" 页顶部显示待审批 Badge）
- 审批范围扩展：不仅是桌面命令，还包括 Agent 策略审批、预算申请、发布审批等
- Agent 进展看板：汇总所有 Agent 的本周 OKR 完成度
- 单 Agent 详情页：查看该 Agent 的产出、提出修改意见
