## 2. 语调指南 (Tone of Voice)

| 场景 | 风格 |
|------|------|
| 开发者文档、API、白皮书 | 正式、严谨，可用术语 |
| 落地页、市场、新闻 | 简洁直白，用"人话"解释技术 |
| 社交媒体、社区 | 亲切轻松，适度幽默 |
| 安全、合规、金融 | 必须严肃 |

### 中英双语风格
- **英文**: 简洁直接，美式科技风，强调创新与开放
- **中文**: 专业但不生硬，注重逻辑与温度，避免官话套话

## 3. 写作规范

### Do's & Don'ts

| ✅ 推荐 | ❌ 避免 |
|---------|--------|
| 主动语态 | 被动语态泛滥 |
| 直接点明价值 | 空洞口号 |
| 解释术语 | 只堆砌技术词 |
| 以用户为中心 | 只谈自己 |
| 适度幽默 | 过度玩梗 |

### 示例对比

✅ **好**: "Agentrix 让你的 AI Agent 不止拥有钱包，更能自主经营、自由交易。"

❌ **坏**: "我们的平台为 AI Agent 提供了多种功能，欢迎体验。"

### 标题规范
- 简洁有力，突出核心价值
- 可用数字、动词增强吸引力
- 避免冗长堆砌

### CTA 规范
- 动词开头，明确行动
- 例：立即体验 / Get Started / Try for Free

## 5. 品牌关键词库

### ✅ 总是使用
AI Agent · Economy · Autonomous · Trade · Growth · Open · Ecosystem · Skills · Marketplace · Platform

### ❌ 避免使用
机器人(Robot) · 黑科技 · 万能 · 颠覆一切 · 革命性（滥用）

### 竞品提及规则
- 可客观对比（Coinbase, OpenAI, CrewAI），但**不贬低**
- 强调 Agentrix 的"经济平台"差异化
- 不直接引用竞品口号
