## 3. Phase 1 增长实验（4月）

### E1: Show HN Launch ⭐ P0

- **假设**: "11 AI Agents 运营一家公司" 足够新奇能上首页
- **标题**: "Show HN: We let 11 AI agents run our AI startup"
- **时机**: 4月第2周 周二 9:00 UTC
- **预期**: 1,000+ 访客 → 50+ 注册
- **成本**: $0

### E2: Twitter/X 内容引擎 ⭐ P0

- **方法**: 每日1条 + 每周2条 Thread
- **内容柱**: Agent 运营日记、技术洞察、行业点评
- **预期**: 4月 +500 Followers, 10K+ 周Impressions
- **成本**: $0

### E3: 开源 Commerce Skills SDK — P1

- **方法**: 开源核心 Skill SDK
- **预期**: 4月 200+ GitHub Stars
- **成本**: $0

### E4: Reddit/IndieHackers 渗透 — P1

- **方法**: r/artificial, r/SideProject + IndieHackers 发帖
- **预期**: 3+ 帖子 50+ upvotes
- **成本**: $0

### E5: Agent Team Early Access Waitlist — P1

- **方法**: 设置 Waitlist 页面收集邮箱
- **预期**: 100+ 注册
- **成本**: $0
