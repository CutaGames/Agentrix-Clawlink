# PayMind Agent 快速开始指南

## 🚀 快速开始

### 1. 启动服务

确保后端和前端服务都在运行：

```bash
# 在项目根目录
bash start.sh

# 或分别启动
cd backend && npm run start:dev
cd paymindfrontend && npm run dev
```

### 2. 访问Agent页面

打开浏览器访问：
```
http://localhost:3000/agent
```

## 📖 功能使用指南

### 对话功能

1. **商品浏览**
   - 输入："浏览商品" 或 "我想买笔记本电脑"
   - Agent会引导您到商品市场

2. **订单查询**
   - 输入："查询订单" 或 "我的订单"
   - Agent会显示订单信息

3. **代码生成**
   - 输入："生成支付代码" 或 "如何创建支付"
   - Agent会生成API调用示例

4. **操作引导**
   - 输入："如何注册" 或 "如何接入API"
   - Agent会提供详细步骤

### 商品市场

1. 点击顶部"商品市场"按钮
2. 使用分类筛选商品
3. 点击商品查看详情
4. 点击商品触发支付流程

### 代码生成

1. 点击顶部"代码生成"按钮
2. 选择编程语言（TypeScript/JavaScript/Python）
3. 在对话中输入需求，或直接输入描述
4. 查看生成的代码示例
5. 点击"复制代码"按钮复制到剪贴板

## 🔧 开发调试

### 查看API响应

打开浏览器开发者工具，查看Network标签页，可以看到：
- `/api/agent/chat` - 对话API调用
- `/api/agent/generate-code` - 代码生成API调用
- `/api/products` - 商品列表API调用

### 后端日志

查看后端日志：
```bash
tail -f logs/backend.log
```

### 前端日志

查看浏览器控制台，可以看到前端日志和错误信息。

## 📝 常见问题

### Q: Agent没有响应？
A: 检查后端服务是否正常运行，查看后端日志。

### Q: 商品列表为空？
A: 确保数据库中有商品数据，或使用商户后台创建商品。

### Q: 代码生成不准确？
A: 当前版本使用规则匹配，后续会集成AI模型提升准确性。

### Q: 支付流程无法完成？
A: 确保已登录并配置了支付方式。

## 🎨 自定义配置

### 修改Agent响应

编辑 `backend/src/modules/agent/agent.service.ts`，修改 `processMessage` 方法。

### 添加新的代码示例

编辑 `components/agent/CodeGenerator.tsx`，在相应方法中添加新的代码模板。

### 修改UI样式

所有组件使用Tailwind CSS，可以直接修改className。
