# Agentrix V3.0 快速测试指南

## 🚀 5分钟快速验证

### 1. 启动服务（2分钟）

```bash
# WSL/Linux
./启动所有服务-V3.sh

# Windows
启动所有服务-V3.bat
```

### 2. 验证服务运行（30秒）

打开浏览器访问：
- ✅ 前端: http://localhost:3000
- ✅ 后端API: http://localhost:3001/api/health
- ✅ API文档: http://localhost:3001/api/docs

### 3. 快速功能测试（2分钟）

#### 测试1: Agent商品搜索
1. 访问 http://localhost:3000/agent
2. 输入: `帮我找一把游戏剑`
3. ✅ 应该返回商品列表

#### 测试2: Agent自动下单
1. 输入: `帮我把火焰之刃加入购物车并支付`
2. ✅ 应该创建订单

#### 测试3: 支付系统PayIntent
访问API文档: http://localhost:3001/api/docs
1. 找到 `/api/payments/payintent` POST接口
2. 创建PayIntent
3. ✅ 应该返回payUrl

## 🆘 常见问题

### 问题1: 后端服务无法启动
**解决**: 检查数据库连接，运行 `npm run migration:run`

### 问题2: 商品搜索无结果
**解决**: 运行 `npm run seed:products` 导入商品数据

### 问题3: 前端页面无法访问
**解决**: 检查前端服务是否运行，端口3000是否被占用
