## 触发机制

### 自动触发 (Cron)

| Agent | 频率 | Cron 表达式 |
|-------|------|-------------|
| ARCHITECT-01 | 每 15 分钟 | `*/15 * * * *` |
| GROWTH-01 | 每 20 分钟 | `*/20 * * * *` |
| BD-01 | 每 30 分钟 | `*/30 * * * *` |
| RESOURCE-01 | 每 5 分钟 | `*/5 * * * *` |
| OPS-01 | 每 30 分钟 | `*/30 * * * *` |
| CONTENT-01 | 每 60 分钟 | `0 * * * *` |

### 按需触发

| Agent | 触发条件 |
|-------|----------|
| DEV-01 | ARCHITECT-01 分配任务时 |
| 所有 Agent | 老板手动指令 |
| 所有 Agent | 紧急事件 |
