# PayMind Agent V3.0 新界面功能测试指南

## 🎯 测试目标

验证新优化的 PayMind Agent 界面是否能够：
1. 正确显示三栏布局（左侧功能面板 + 中间主内容）
2. 展示 Agent 能力和身份信息
3. 支持结构化消息卡片
4. 显示工作流记录
5. 实现自动比价和推荐功能

## 📋 测试步骤

### 1. 访问界面
- 打开浏览器访问：`http://localhost:3000/agent`
- 确认页面正常加载，无错误提示

### 2. 检查布局
- ✅ **左侧功能面板**：应显示 Agent 能力列表、身份信息、快捷操作
- ✅ **顶部导航**：应显示图标化的导航按钮（对话、商品市场、购物车等）
- ✅ **中间主内容**：应显示聊天界面或选中的功能页面

### 3. 测试聊天功能
- 发送消息："帮我找一款1000元以内的电纸书并下单"
- 检查是否显示：
  - ✅ 结构化商品卡片（图片、价格、按钮）
  - ✅ 价格对比卡片（自动比价）
  - ✅ 工作流记录（可展开查看AI决策过程）

### 4. 测试结构化消息
- 测试商品卡片：点击"加入购物车"或"立即购买"
- 测试订单流程：发送"查询订单"相关消息
- 测试代码生成：发送"生成一个支付 USDT 的 API demo"

### 5. 测试功能面板
- 点击左侧功能面板的各个能力项
- 检查是否正确跳转到对应页面
- 测试快捷操作（创建商品、生成支付链接等）

### 6. 测试导航切换
- 点击顶部导航的不同按钮
- 确认页面正确切换
- 检查购物车数量徽章是否正确显示

### 7. 测试初始提示
- 检查初始欢迎消息是否显示完整的 Agent 能力介绍
- 确认示例按钮可以快速填充输入框

## 🐛 已知问题

## ✅ 预期结果

1. 界面布局清晰，三栏结构完整
2. Agent 身份信息正确显示（运行状态、用户模式、KYC状态）
