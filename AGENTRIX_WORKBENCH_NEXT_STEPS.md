# Agentrix 工作台后续开发计划

**日期**: 2026-01-05  
**时间线**: 2026年1月 - 2026年2月

---

## 1. 第一阶段：功能恢复与 API 重新挂载 (第 1-2 周)
**目标**: 确保 V3 UI 架构下的功能完整性，对齐 2025.12.29 版本的可用性。

- **任务**:
  - [ ] **用户端恢复**: 将 `UserModule.tsx` (V1) 中的支付历史、钱包管理、KYC 和订单跟踪逻辑迁移至 `UserModuleV2.tsx`。
  - [ ] **商户端恢复**: 将 `MerchantModule.tsx` (V1) 中的商品 CRUD、Shopify 同步、结算中心和数据分析逻辑迁移至 `MerchantModuleV2.tsx`。
  - [ ] **开发者端恢复**: 在 `DeveloperModuleV2.tsx` 中重新挂载 `SkillRegistry`、`PackCenter` 和 `TestHarness` 组件。
  - [ ] **导航对齐**: 确保 `UnifiedWorkspace.tsx` 的 L1/L2 导航能正确触发各模块的数据加载。

**交付物**:
- 功能完整的 `UserModuleV2`, `MerchantModuleV2`, `DeveloperModuleV2`。
- 恢复 12.29 版本所有已打通的 API 调用。

## 2. 第二阶段：UI/UX 深度优化 (第 3-4 周)
**目标**: 在功能恢复的基础上，完成 V3.0 信息架构的视觉与交互升级。

- **任务**:
  - [ ] **侧边栏工具箱**: 实现可折叠的 L2 侧边栏，整合 Agent 管理、技能市场和安全审计。
  - [ ] **响应式适配**: 优化新架构在移动端和不同屏幕尺寸下的表现。
  - [ ] **交互增强**: 为技能打包、商品转换等复杂操作添加进度条和实时反馈。

**交付物**:
- 符合 V3.0 设计规范的统一工作台界面。
- 优化的移动端交互体验。

## 3. 第三阶段：技能生态与商业闭环 (第 5-6 周)
**目标**: 增强 Agent Builder 能力，深化电商集成。

- **任务**:
  - [ ] **多平台分发**: 增强 `PackCenter`，支持 Gemini 1.5 Pro 和 Grok 的最新工具调用 Schema。
  - [ ] **Shopify 深度集成**: 实现订单状态双向同步和 AI 驱动的商品描述生成。
  - [ ] **技能市场**: 开放技能市场预览，支持开发者一键发布技能。

**交付物**:
- 增强版技能打包器。
- 深度集成的商户电商模块。

## 4. 第四阶段：安全、审计与合规 (第 7-8 周)
**目标**: 建立透明的 Agent 运行环境，确保用户信任。

- **任务**:
  - [ ] **人类可读审计**: 完成所有链上/API 操作的自然语言翻译引擎。
  - [ ] **MPC 策略增强**: 在安全中心增加基于时间、额度和地点的 MPC 支付策略配置。
  - [ ] **合规报告导出**: 支持商户和用户导出加密签名的审计报告。

**交付物**:
- 人类可读的审计中心。
- 高级 MPC 策略管理器。

---

## 2026 Q1 交付物汇总
1. **统一工作台 V3.0**: 角色化、任务导向的专业界面，功能 100% 对齐并超越 12.29 版本。
2. **多平台技能打包器**: 支持 OpenAI, Claude, Gemini, Grok。
3. **Agent 商业引擎**: 从商品同步到 Agent 自动下单的全链路闭环。
4. **透明审计系统**: 所有 Agent 活动均有据可查，人类可读。
