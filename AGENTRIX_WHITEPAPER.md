# 1. 执行摘要

## 1.1 定位
**Agentrix = AI Agent 时代的 Stripe + App Store**

我们为 AI Agent 生态提供：
- **支付基础设施**：Agent-to-Agent、Agent-to-Merchant 的价值流转
- **分佣结算引擎**：多方协作的自动化收益分配
- **Skill 市场**：AI 能力的发现、交易与货币化

## 3.2 Commerce Skill 详解

三种使用模式：

| 模式 | 场景 | 费率 |
|------|------|------|
| `PAY_ONLY` | 纯收款 | 0-0.1% |
| `SPLIT_ONLY` | 外部订单分账 | 0.3% |
| `PAY_AND_SPLIT` | 收款+自动分账 | 0.4% |

**费率公式**：
```
总费用 = Onramp费(0.1%) + Offramp费(0.1%) + 分佣费(0.3%)
纯 Crypto 支付：0% ✨
```

  amount: 100,

# 5. Token 经济学 (AGX)

## 5.1 设计原则

1. **刚需消耗**：Token 必须被"用掉"而非只是"持有"
2. **供需平衡**：释放速度与生态增长挂钩
3. **价值锚定**：Token 价值与平台 GMV 强关联
4. **博弈均衡**：质押者、消费者、贡献者利益对齐

## 5.2 Token 基本参数

| 参数 | 值 |
|------|-----|
| 名称 | AGX (Agentrix Token) |
| 总量 | 1,000,000,000 (10亿，固定) |
| 初始流通 | 5% (TGE 解锁) |
| 链 | Base (主) + Arbitrum + Solana |
| 标准 | ERC-20 / SPL |

## 5.3 Token 效用（7 大消耗场景）

### 🔥 消耗型效用（Burn/Lock）

| 效用 | 机制 | 消耗强度 |
|------|------|---------|
| **1. 手续费折扣** | 用 AGX 支付手续费享 50% 折扣，AGX 销毁 | ⭐⭐⭐⭐⭐ |
| **2. Skill 上架质押** | 发布 Skill 需质押 100-10,000 AGX | ⭐⭐⭐⭐ |
| **3. 优先推荐权** | 支付 AGX 获得 Skill 市场首页推荐 | ⭐⭐⭐ |
| **4. 企业 API 额度** | 超出免费额度需 AGX 解锁 | ⭐⭐⭐ |

### 💎 治理型效用（Lock/Vote）

| 效用 | 机制 |
|------|------|
| **5. veAGX 治理** | 锁仓 AGX → veAGX，参与生态基金投向、费率调整投票 |
| **6. 收益加成** | veAGX 持有者获得 Staking 收益加成（最高 2.5x） |
| **7. 生态基金分红** | veAGX 持有者按比例分享平台收入的 20% |

## 5.5 动态释放机制（核心创新）

**问题**：固定释放导致供过于求，币价下跌

**解决方案**：**生态增长挂钩释放**

```
月释放量 = 基础释放 × 增长系数

增长系数 = min(2.0, max(0.5, 当月GMV / 上月GMV))

示例：
- GMV 增长 50%：释放量 × 1.5
- GMV 下降 20%：释放量 × 0.8（保护币价）
- GMV 翻倍：释放量 × 2.0（激励增长）
```

## 5.6 价值捕获机制

### 回购销毁（Buyback & Burn）

| 来源 | 回购比例 | 销毁比例 |
|------|---------|---------|
| 交易手续费 | 30% | 50% 销毁，50% 激励池 |
| Skill 市场抽成 | 20% | 100% 销毁 |
| 企业订阅 | 10% | 100% 激励池 |

**量化承诺**：
```
年度回购金额 = 年收入 × 20%

Y1: $2M × 20% = $400K 回购
Y2: $25M × 20% = $5M 回购
Y3: $100M × 20% = $20M 回购
```

### 价值锚定公式

```
AGX 理论估值 = (年 GMV × Take Rate × 收益倍数) / 流通量

示例（Y3）：
= ($2B × 1.5% × 20) / 500M 流通
= $1.2 / AGX
```

## 5.7 Staking 与 veAGX

### veAGX 机制

```
veAGX 数量 = AGX 数量 × 锁定时间系数

锁定时间系数：
- 1 周：0.1x
- 1 个月：0.25x
- 6 个月：0.5x
- 1 年：1.0x
- 4 年：4.0x（最大）
```

### Staking 收益

| Skill 质押 | 按 Skill 收益 | 按 Skill 等级 |
