## 第一部分：Commerce Skill 商业引擎升级

### 1. 核心架构与三种支付模式
Commerce Skill 现在支持完整的商业闭环，涵盖从法币支付到链上分账的全流程。
- **PAY_ONLY**: 传统的支付模式，资金进入商户账户。
- **SPLIT_ONLY**: 纯分账模式，适用于已有资金进入系统的场景，通过智能合约执行分润。
- **PAY_AND_SPLIT (核心亮点)**: 支付与分账同步进行。支持 Stripe/PayPal 等法币入金后，自动触发链上分佣规则，实现“结算即分配”。

### 2. 支付集成的技术进步
- **幂等性检查 (Idempotency)**: 引入强效的幂等控制机制，防止重复支付与双重结算。
- **多币种结算**: 支持 USD 与加密货币（USDC/USDT）的自动转换与清算。
- **多端对账**: 实现了法币订单状态、支付会话与区块链交易状态的三方同步。

### 3. 分润规则的可视化
- **预览面板 (CommissionPreviewPanel)**: 在交易发生前，为商户和 Agent 提供透明的费用分解展示。
- **多级分润支持**: 支持商户手续费、Agent 基础分佣、平台抽成及外部（如推荐人）引流奖励的复杂计算。

## 第二部分：Agentrix 移动端 App 产品规划 (PRD 核心)

### 1. 核心定位：Agent 的掌上指挥部
移动端 App 不仅仅是一个管理后台，它是 Agent 参与生态协作、管理收益的移动枢纽。

### 2. 三大核心基础
1. **统一资金账户 (Unified Fund Account)**: 
   - 整合法币余额与加密货币资产。
   - 实现“一键结算”与灵活的子账户管理，解决 Agent 跨境收付款难题。
2. **Agent 账户体系 (Agent Identity)**:
   - 移动端承载 Agent 身份（Profile），包括能力评级、历史收益与任务表现。
   - 建立信用体系，作为参与高额预算池（Budget Pool）的敲门砖。
3. **社交与增长 (Social & Growth)**:
   - **一键分享分佣方案**: Agent 之间可以像发名片一样分享分佣规则。
   - **推荐链接 (Referral)**: 内置社交分享机制，通过引流自动获取系统佣金分配。

### 3. 主打功能 (Killer Features)
- **实时收益看板**: 实时推送分佣到账、预算池释放提醒。
- **轻量审批流**: 移动端针对里程碑（Milestones）的快速提交与一键确认审批。
- **任务市场 (Task Market)**: 移动优先的任务抢单与协作模块。
