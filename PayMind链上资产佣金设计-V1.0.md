## 1. 链上资产交易场景分析

### 1.1 场景分类

#### 1.1.1 按资产类型分类

**NFT（Non-Fungible Token）**：
- 数字艺术品
- 游戏道具
- 收藏品
- 域名
- 会员卡
- 其他NFT资产

**FT（Fungible Token）**：
- 代币（Token）
- 稳定币（USDC、USDT等）
- 治理代币
- 其他FT资产

**游戏资产（Game Assets）**：
- 游戏道具NFT（武器、角色、装备、皮肤等）
- 游戏代币（游戏内货币、治理代币、实用代币）
- 游戏内虚拟资产（土地、建筑等）

**RWA（现实世界资产）**：
- 代币化资产（房地产、商品、股票、债券等）
- 链上凭证（学历证书、职业证书、身份凭证、所有权凭证等）
- 收益代币（Staking、Lending、Yield Farming等）

#### 1.1.2 按交易场景分类

**场景1：平台早期聚合交易**
- PayMind平台早期聚合的NFT/FT/游戏资产/RWA交易

**场景3：Launchpad（项目发行）**

**场景4：二级市场交易**

**场景5：跨链交易**

- 平台提供跨链桥

**场景6：DEX聚合交易**

- 平台提供聚合服务
- 平台承担聚合风险

**场景7：游戏资产交易**
- 游戏道具NFT交易
- 游戏代币交易
- 游戏内虚拟资产交易

**场景8：RWA（现实世界资产）交易**

## 2. 佣金设计原则

### 2.1 设计原则

**原则1：场景差异化**
- 不同场景不同佣金比例
- 平台聚合场景佣金较高（平台承担更多成本）
- 用户自发场景佣金较低（用户承担更多成本）

**原则2：成本匹配**
- 佣金比例匹配平台成本
- 平台聚合场景：佣金覆盖流动性、运营、风险成本
- 用户自发场景：佣金覆盖基础设施、撮合、技术支持成本

**原则3：竞争力**
- 参考主流平台佣金比例
- 保持市场竞争力
- 吸引用户和项目方

**原则4：生态激励**
- 激励用户使用平台聚合
- 激励项目方在平台发行
- 激励用户自发交易

## 4. 平台聚合 vs 用户自发

### 4.1 区别分析

#### 4.1.1 平台聚合交易

**特点**：
- 平台提供流动性
- 平台提供撮合服务
- 平台承担风险
- 平台维护和运营

**成本**：
- 流动性成本（高）
- 运营成本（高）
- 风险成本（高）
- 技术成本（中）

**佣金比例**：
- NFT：**2.5%**（PayMind 1% + 其他 1.5%）
- FT：**0.3%**（PayMind 0.1% + 其他 0.2%）

#### 4.1.2 用户自发交易

**特点**：
- 用户自己发行资产
- 用户自己提供流动性（可选）
- 平台提供撮合服务
- 平台提供基础设施

**成本**：
- 基础设施成本（中）
- 撮合服务成本（中）
- 风险控制成本（低）
- 技术支持成本（低）

**佣金比例**：
- NFT：**1.5%**（PayMind 0.5% + 其他 1%）
- FT：**0.15%**（PayMind 0.05% + 其他 0.1%）

### 4.2 佣金差异原因

#### 4.2.1 成本差异

**平台聚合**：
- 需要提供流动性池（成本高）
- 需要维护流动性（成本高）
- 需要承担风险（成本高）
- 佣金比例较高

**用户自发**：
- 用户自己提供流动性（成本低）
- 平台只提供撮合（成本低）
- 用户承担风险（成本低）
- 佣金比例较低

#### 4.2.2 价值差异

**平台聚合**：
- 为用户提供流动性（价值高）
- 降低交易成本（价值高）
- 提升交易效率（价值高）
- 保障交易安全（价值高）

**用户自发**：
- 用户自主发行（价值中）
- 平台提供基础设施（价值中）
- 平台提供撮合（价值中）
- 用户承担风险（价值低）

### 4.3 激励策略

#### 4.3.1 平台聚合激励

**激励措施**：
- 提供流动性奖励

- 按资产来源分别计算佣金

- 平台提供批量交易服务

**佣金设计**：
- 按资产数量计算佣金

### 7.3 限时活动

#### 7.3.1 场景：平台限时活动

- 例如：NFT交易佣金从2.5%降至1.5%（折扣40%）

#### 8.2.1 佣金计算服务

```typescript
@Injectable()
export class OnChainAssetCommissionService {
  /**
   * 计算链上资产交易佣金
   */
  async calculateCommission(
    assetType: 'nft' | 'ft',
    transactionType: 'platform_aggregated' | 'user_generated' | 'launchpad' | 'cross_chain' | 'dex_aggregated',
    amount: number,
    projectType?: 'premium' | 'standard' | 'high_risk',
    crossChainType?: 'same_type' | 'different_type',
  ): Promise<CommissionCalculation> {
    // 1. 确定佣金比例
    const commissionRate = this.getCommissionRate(
      assetType,
      transactionType,
      projectType,
      crossChainType,
    );
    
    // 2. 计算PayMind平台费
    const paymindFeeRate = this.getPayMindFeeRate(
      assetType,
      transactionType,
      projectType,
    );
    const paymindFee = amount * paymindFeeRate;
    
    // 3. 计算其他渠道佣金
    const otherChannelFeeRate = commissionRate - paymindFeeRate;
    const otherChannelFee = amount * otherChannelFeeRate;
    
    // 4. 计算卖家收入
    const sellerIncome = amount - paymindFee - otherChannelFee;
    
    return {
      totalCommission: amount * commissionRate,
      paymindFee,
      paymindFeeRate,
      otherChannelFee,
      otherChannelFeeRate: otherChannelFeeRate,
      sellerIncome,
      commissionRate,
    };
  }
  
  /**
   * 获取佣金比例
   */
  private getCommissionRate(
    assetType: 'nft' | 'ft',
    transactionType: string,
    projectType?: string,
    crossChainType?: string,
  ): number {
    // NFT场景
    if (assetType === 'nft') {
      if (transactionType === 'platform_aggregated') {
        return 0.025; // 2.5%
      }
      if (transactionType === 'user_generated') {
        return 0.015; // 1.5%
      }
      if (transactionType === 'launchpad') {
        if (projectType === 'premium') {
          return 0.05; // 5%
        }
        if (projectType === 'standard') {
          return 0.06; // 6%
        }
        if (projectType === 'high_risk') {
          return 0.08; // 8%
        }
      }
    }
    
    // FT场景
    if (assetType === 'ft') {
      if (transactionType === 'platform_aggregated') {
        return 0.003; // 0.3%
      }
      if (transactionType === 'user_generated') {
        return 0.0015; // 0.15%
      }
      if (transactionType === 'launchpad') {
        if (projectType === 'premium') {
          return 0.05; // 5%
        }
        if (projectType === 'standard') {
          return 0.06; // 6%
        }
        if (projectType === 'high_risk') {
          return 0.08; // 8%
        }
      }
      if (transactionType === 'dex_aggregated') {
        return 0.002; // 0.2%
      }
    }
    
    // 跨链场景
    if (transactionType === 'cross_chain') {
      if (crossChainType === 'same_type') {
        return 0.005; // 0.5%
      }
      if (crossChainType === 'different_type') {
        return 0.01; // 1%
      }
    }
    
    // 默认
    return 0.01; // 1%
  }
  
  /**
   * 获取PayMind平台费率
   */
  private getPayMindFeeRate(
    assetType: 'nft' | 'ft',
    transactionType: string,
    projectType?: string,
  ): number {
    // NFT场景
    if (assetType === 'nft') {
      if (transactionType === 'platform_aggregated') {
        return 0.01; // 1%
      }
      if (transactionType === 'user_generated') {
        return 0.005; // 0.5%
      }
      if (transactionType === 'launchpad') {
        if (projectType === 'premium') {
          return 0.01; // 1%
        }
        if (projectType === 'standard') {
          return 0.015; // 1.5%
        }
        if (projectType === 'high_risk') {
          return 0.02; // 2%
        }
      }
    }
    
    // FT场景
    if (assetType === 'ft') {
      if (transactionType === 'platform_aggregated') {
        return 0.001; // 0.1%
      }
      if (transactionType === 'user_generated') {
        return 0.0005; // 0.05%
      }
      if (transactionType === 'launchpad') {
        if (projectType === 'premium') {
          return 0.01; // 1%
        }
        if (projectType === 'standard') {
          return 0.015; // 1.5%
        }
        if (projectType === 'high_risk') {
          return 0.02; // 2%
        }
      }
      if (transactionType === 'dex_aggregated') {
        return 0.001; // 0.1%
      }
    }
    
    // 跨链场景
    if (transactionType === 'cross_chain') {
      return 0.002; // 0.2%（基础，根据跨链类型调整）
    }
    
    // 默认
    return 0.01; // 1%
  }
}
```

3. **成本匹配**：佣金比例匹配平台成本
4. **市场竞争力**：参考主流平台，保持竞争力

### 9.2 佣金比例总结

**实体商品**：3%（PayMind 0.5% + 其他 2.5%）

**平台聚合NFT**：2.5%（PayMind 1% + 其他 1.5%）
**用户自发NFT**：1.5%（PayMind 0.5% + 其他 1%）

**Launchpad**：5-8%（根据项目类型）

**DEX聚合**：0.2%（PayMind 0.1% + 其他 0.1%）
