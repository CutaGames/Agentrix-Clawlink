# 修复后台管理服务启动问题

## 问题
`ts-node` 不支持 `--watch` 选项，导致启动失败。

## 解决方案

### 方案1: 安装nodemon（推荐）

```bash
cd backend
npm install --save-dev nodemon
```

然后运行：
```bash
npm run start:admin:dev
```

### 方案2: 直接启动（无热重载）

如果不需要热重载，可以直接使用：
```bash
npm run start:admin
```

### 方案3: 安装ts-node-dev

```bash
cd backend
npm install --save-dev ts-node-dev
```

然后修改 `package.json` 中的脚本为：
```json
"start:admin:dev": "ts-node-dev --respawn --transpile-only src/main-admin.ts"
```

## 已修复

已更新 `package.json`：
- 添加 `nodemon` 依赖
- 修改 `start:admin:dev` 脚本使用 `nodemon`

现在可以运行：
```bash
npm install  # 安装nodemon
npm run start:admin:dev  # 启动开发模式
```
