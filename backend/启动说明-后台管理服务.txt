==========================================
PayMind 后台管理服务启动说明
==========================================

问题解决：
1. 你已经在backend目录下，不需要再执行 cd backend
2. ts-node-dev未安装，已改为使用ts-node（已包含在依赖中）

启动方式：

方式1: 直接启动（推荐）
----------------------------------------
npm run start:admin

方式2: 开发模式（支持热重载）
----------------------------------------
npm run start:admin:dev

注意：
- 确保在backend目录下执行
- 如果端口3002被占用，可以设置环境变量：
  ADMIN_PORT=3003 npm run start:admin

启动成功后访问：
- API文档: http://localhost:3002/api/docs
- 健康检查: http://localhost:3002/api/health
