# Agentrix RAG 知识库测试文件

欢迎使用 Agentrix 总部本地 RAG 系统！

## 如何使用 RAG？
1. 将你的 PDF、Markdown 或 TXT 文件放入 `backend/knowledge/` 目录。
2. 当你在 HQ 控制台中向 Agent 提问时，如果它们需要更深度的背景信息，会自动调用 `search_local_docs` 工具检索该目录下的内容。

## 当前项目关键信息
- **项目代号**: Agentrix V7
- **核心协议**: UCP (Unified Commerce Protocol) 和 X402 (Payment Protocol)。
- **研发目标**: 实现 AI Agent 之间的零摩擦商业协作。
